- Initial release of DataInsights AI platform
- Agentic AI architecture using LangGraph
- Natural language to SQL conversion
- Multi-database support (PostgreSQL, MySQL, SQL Server, SQLite, MongoDB)
- Real-time query execution
- Statistical analysis and pattern detection
- Automatic visualization recommendations
- JWT authentication
- Redis caching
- Rate limiting
- WebSocket support for real-time updates
- Modern Next.js 14 frontend with TypeScript

- Comprehensive documentation
